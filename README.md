# VantaVoice
## 1. Introduction
VantaVoice is a sophisticated, voice-first AI assistant application designed to run natively on Windows. 
Initially conceived as a co-pilot for Windows PC Users and Video Gamers, the application has evolved into a versatile, 
extensible platform that integrates multiple Large Language Models (LLMs) and advanced features like screen capture for contextual awareness.

## Repository
This repository created to distribute releases of the VantaVoice client application.
